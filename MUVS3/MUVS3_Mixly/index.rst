MU 3 Mixly 教程
===============================

本文介绍MU Vision Sensor 3 配合Arduino开发板在米思奇（Mixly）开发环境下的开发教程。

Mixly是一款图形化编程软件，用户可以通过拼接积木块的方式来编写程序。

Mixly基础教程请参考官方帮助文档 `Mixly 帮助文档 <http://mixly.readthedocs.io/>`_
